## 二、★ 正确方案: 候选C(patch isHungry setter)

guard patch(0x466e2c/0x466e28)都不行(共享函数). 正确方案 = **不改 guard, 改 isHungry 的 setter**:
- isHungry(sing+0x20) 的 TRUE 写点(setter) = 每次算完"猫饿"后写 TRUE
- 找到 setter → patch 恒写 FALSE → isHungry 永远 FALSE → guard 自然走 FALSE 分支(不塞猫)
- **不动共享函数, 只改 setter**

### setter候选(主控给的5个str[#0x20])
```
0x46238c - str [x26,#0x20] (thread+0x20? 可能不是singleton+0x20)
0x46242c - str [x26,#0x20]
0x466f48 - str [x1,#0x20]
0x466f64 - str [x3,#0x20]
0x466fb8 - str [x1,#0x20]
```
0x466f28 含 FALSE setter(add x0,x22,#0x20; str [x1,#0x20])
